## Architecture Notes

- **Auto-generated types** (`Types/` directory): Generated by UA Model Compiler from IJT NodeSet files. Never edit manually — regenerate when NodeSets change.
- **UAModel namespace**: All generated type bindings live under `UAModel.*` — excluded from coverage measurement.
- **OPC UA SDK**: OPC Foundation UA .NET Standard (`Opc.Ua.Core`, `Opc.Ua.Client`) — industry standard, not third-party.
- **Interactive menu**: `Program.cs` drives the menu loop; each management class handles one domain.
